# Tech News

A modern web application for browsing and reading tech news articles. Built with React for the frontend and Node.js/Express for the backend.

## Features

- Browse latest tech news articles
- Filter articles by category
- View detailed article pages
- Author profiles
- User profiles and account management
- Responsive design with Material-UI
- Scroll to top functionality

## Tech Stack

### Frontend
- React 18
- Material-UI (@mui/material)
- React Router DOM
- Axios for API calls

### Backend
- Node.js
- Express.js
- CORS
- Dotenv for environment variables
- Nodemailer for email functionality
- Multer for file uploads

## Installation

### Prerequisites
- Node.js >= 14.0.0
- npm or yarn

### Frontend Setup
1. Clone the repository
2. Navigate to the root directory
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file with necessary environment variables
4. Start the backend server:
   ```bash
   npm run dev
   ```

## Usage

- Visit the home page to see the latest news
- Use the navbar to navigate to different categories
- Click on articles to read full content
- Access user profile and account pages
